Instalación
===========

1) Correr database.sql como root del mysql

2) Correr ddl.sql DJMAG_USER/DJMAG_USER base DJMAG

3) Copiar el war al directorio webapps del tomcat (ej: apache-tomcat-6.0.32/webapps)

4) Copiar el archivo DJMAG_WEB.xml al directorio de contextos del tomcat (ej: apache-tomcat-6.0.32/conf/Catalina/localhost)


Accesos
=======

Sitio público: ./index.jsp

BackOffice: ./boLogin.jsp

Cambio de password del Admin
UPDATE SYSTEMUSER SET password = SHA1('Admin') WHERE username = 'Admin';
COMMIT;
